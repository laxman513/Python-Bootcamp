# 🧠 Java vs Python

### Java

```java
public static int add(int a, int b){
    return a + b;
}
```

### Python

```python
def add(a, b):
    return a + b
```

Python is simpler because it doesn't require:

- public
- private
- static
- return type
- braces `{}`

# ⚠ Common Mistakes

- Forgetting to call a function.
- Confusing `print()` with `return`.
- Accessing local variables outside a function.
- Passing positional arguments after keyword arguments.
- Forgetting that `*args` is a tuple.
- Forgetting that `**kwargs` is a dictionary.

---

# 🏆 Best Practices

- Use meaningful function names.
- Keep functions short and focused.
- Return values instead of printing whenever appropriate.
- Avoid unnecessary global variables.
- Use keyword arguments for better readability.
- Write reusable functions.
